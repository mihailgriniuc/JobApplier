Privacy Policy

Last updated: March 2026

Key Point: Your data is stored locally on your device by default. If you explicitly enable AI answers, the extension sends the relevant prompt context to Mistral using the API key you provide.

Data Collection

Job Application Autofill collects the following information that you voluntarily provide:

  • Personal information (name, email, phone, LinkedIn URL, location)
  • Work authorization status
  • Demographic information (for EEOC compliance)
  • Resume document

Data Storage

All data is stored locally on your device using Chrome's built-in storage API. Your data:

  • Remains on your device unless you enable the optional AI-answer feature
  • Is sent to Mistral only when AI answers are enabled and only for the context needed to generate an answer
  • Is not shared with third parties
  • Is not used for analytics or tracking

Data Usage

Your data is used solely to:

  • Auto-fill job application forms when you trigger the autofill feature
  • Display your saved information in the extension popup and options page
  • Generate optional AI answers for open-ended screening questions when you enable that feature

Optional AI Answers

If you enable AI answers in Settings, the extension sends the following prompt context to Mistral:

  • The open-ended question text
  • Relevant job-posting text from the current page
  • Your saved profile fields that help answer the question
  • Any extra resume context you explicitly add in Settings

The Mistral API key is stored locally in Chrome storage and used only to call Mistral on your behalf.

Data Security

Your data is protected by Chrome's built-in security mechanisms. The extension does not have access to any data from other websites or browser activities beyond the job application pages where you activate it.

Data Control

You have full control over your data:

  • View: See all stored data in the Options page
  • Edit: Modify any information at any time
  • Export: Download your data as a JSON file
  • Delete: Permanently remove all data with one click

Permissions

This extension requires the following permissions:

  • Storage: To save your data locally
  • Active Tab: To fill forms on the current page
  • Scripting: To detect and interact with form fields

Children's Privacy

This extension is not designed for use by children under 13 years of age.

Changes to This Policy

Any changes to this privacy policy will be reflected in an updated version of the extension.

Contact

If you have questions about this privacy policy, please contact us through the Chrome Web Store.
